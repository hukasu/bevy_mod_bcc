# bevy_mod_bcc

Asset loader for the *Binary Curve Collection* `.bcc` format.

This repository is intended for use  in the [Bevy](https://bevy.org) game engine, but it
is possible to use only the types and reader by disabling the default features.

# Assets

This repository contains the `cable_work_pattern.bcc` copyrighted by Cem Yuksel
found in [Yarn models](https://www.cemyuksel.com/research/yarnmodels/).
